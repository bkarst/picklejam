The core editorial unit — media area, corner badge, uppercase headline, excerpt and meta.

```jsx
<StoryCard badge="Top Story" title="New Champs Crowned in Austin"
  excerpt="A thrilling final goes the distance at the PPA Tour Austin Open."
  meta="News · 2h ago" image="/photo.jpg" />
```

Composes `Card` + `Badge`. Use `variant="sticker"` for the outlined look. Pass `media` to supply a custom media node instead of an image.
