Chunky uppercase CTA button — use for any primary or secondary action across Pickle Jam surfaces.

```jsx
<Button variant="primary" size="md" iconRight={<Icon name="arrow-right" />}>Read more</Button>
```

Variants: `primary` (hot pink, main action), `accent` (lime, subscribe/positive), `ink` (dark green), `outline`, `ghost`. Sizes: `sm` `md` `lg`. All uppercase, bold Montserrat. Use `full` to stretch, `as="a"` + `href` for links.
